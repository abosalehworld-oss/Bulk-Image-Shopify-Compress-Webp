# 🛒 أداة تحميل وضغط ورفع صور منتجات شوبيفاي بالجملة

## 📋 نظرة عامة

أداة بايثون بواجهة رسومية مصممة لمتخصصي SEO تقوم بـ:

1. **📥 تحميل** جميع صور المنتجات من متجر شوبيفاي
2. **🗜️ ضغط** الصور مع الحفاظ على الجودة (اختياري)
3. **☁️ رفع** الصور تلقائيًا إلى المتجر عبر Shopify API (اختياري)

كل منتج صوره بتنزل في **مجلد منفصل** باسمه على جهازك.

## 🎯 الواجهة الرسومية - شرح مفصل

لما تفتح الأداة، هتلاقي الواجهة فيها الأقسام دي:

### 🎯 القسم 1: اختيار العملية

عندك **4 أوضاع** تختار منهم حسب احتياجك:

| الوضع | متى تستخدمه |
|-------|-------------|
| **📥🗜️ تحميل + ضغط** | أول مرة - عايز تحمّل الصور وتضغطها مرة واحدة |
| **📥 تحميل فقط** | عايز تحمّل الصور بس وتضغطها بنفسك ببرنامج تاني |
| **🗜️ ضغط فقط** | عندك صور على جهازك وعايز تضغطها بالأداة |
| **☁️ رفع فقط** | عندك صور جاهزة (ضغطتها بره) وعايز ترفعها لشوبيفاي |

### 📄 القسم 2: ملف CSV
- اضغط **📂 استعراض** واختر ملف CSV اللي صدّرته من شوبيفاي

### 📁 القسم 3: أماكن الحفظ
- **أنت بتختار المكان!** اضغط 📂 واختر أي مجلد على جهازك
- **مجلد الصور المحملة**: فين الصور هتتحفظ لما تتحمل
- **مجلد الصور المضغوطة**: فين النسخ المضغوطة هتتحفظ
- **مجلد الرفع**: (في وضع "رفع فقط") فين الصور الجاهزة اللي عايز ترفعها

### ⚙️ القسم 4: إعدادات
- **جودة الضغط**: حرّك الشريط (30%-100%). الموصى: **80-85%**
- **خيوط التحميل**: عدد التحميلات المتوازية. الافتراضي **10**

### 🔑 القسم 5: Shopify API
- **اسم المتجر**: اسم متجرك (مثال: `cj0fzh-ca`)
- **Access Token**: التوكن من التطبيق المخصص
- زر **👁** لإظهار/إخفاء التوكن

### أزرار التحكم:
- **🚀 تشغيل**: يبدأ العملية
- **⏹ إيقاف**: يوقف العملية (يكمّل الصورة الحالية وبعدين يقف)
- **📁 فتح مجلد الصور**: يفتح المجلد اللي فيه الصور

## 📥 تصدير ملف CSV من شوبيفاي

### الخطوات:

1. **ادخل لوحة التحكم:**
   ```
   https://admin.shopify.com/store/اسم-متجرك
   ```
   مثال: `https://admin.shopify.com/store/cj0fzh-ca`

2. **اذهب لصفحة المنتجات:**
   - من القائمة الجانبية → **Products**

3. **اضغط Export:**
   - اضغط زر **Export** في أعلى الصفحة

4. **اختر المنتجات:**
   - **All products** = كل المنتجات ✅ (موصى)
   - **Current page** = الصفحة الحالية فقط

5. **اختر الصيغة:**
   - **CSV for Excel, Numbers, or other spreadsheet programs**

6. **اضغط Export products** وحمّل الملف

> ⚠️ **لا تفتح الملف في Excel** قبل ما تستخدمه. Excel ممكن يفسد الترميز.

## 🔑 إعداد Shopify API للرفع التلقائي

> ✅ **مجاني 100%** - مش هيكلفك حاجة
> ✅ **آمن** - بيوصل للمنتجات بس، مش بيانات العملاء أو الطلبات

### الخطوات:

1. **اذهب لإعدادات التطبيقات:**
   ```
   Settings → Apps and sales channels → Develop apps
   ```

2. **فعّل التطوير** (مرة واحدة):
   - اضغط **"Allow custom app development"** ← أكّد

3. **أنشئ تطبيق:**
   - اضغط **"Create an app"**
   - الاسم: `Image Optimizer`
   - اضغط **"Create app"**

4. **حدد الصلاحيات:**
   - اضغط **"Configure Admin API scopes"**
   - فعّل:
     - ✅ `read_products`
     - ✅ `write_products`
   - اضغط **Save**

5. **ثبّت التطبيق:**
   - اضغط **"Install app"** ← **"Install"**

6. **انسخ التوكن:**
   - اضغط **"Reveal token once"**
   - ⚠️ **انسخه فورًا! مش هيظهر تاني**
   - التوكن بيبدأ بـ `shpat_`

### في الأداة:
- **اسم المتجر**: `cj0fzh-ca`
- **Access Token**: الصقه في الخانة

## ❓ أسئلة شائعة

### ❓ لو التحميل فشل في بعض الصور؟
الأداة بتعيد المحاولة 3 مرات. لو فشل تاني، شغّلها مرة كمان - هتتخطى الصور المحملة وتكمّل الباقي.

### ❓ هل أقدر أوقف وأكمل بعدين؟
✅ أيوه! اضغط ⏹ إيقاف. لما تشغّلها تاني هتكمّل من حيث وقفت.

### ❓ هل الأداة آمنة على المتجر؟
✅ آمنة تمامًا:
- بتطلب تأكيدك قبل أي رفع
- بتحتفظ بالصور الأصلية على جهازك
- بتحترم حدود طلبات شوبيفاي

### ❓ أقدر أضغط الصور ببرنامج تاني؟
✅ طبعًا! حمّل بوضع "تحميل فقط"، اضغط الصور بأي برنامج، وارجع ارفعها بوضع "رفع فقط".

### ❓ أقدر أستخدمها مع أكثر من متجر؟
✅ غيّر اسم المتجر والتوكن في الخانات وشغّلها.
